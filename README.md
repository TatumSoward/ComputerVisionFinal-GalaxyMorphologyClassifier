# ComputerVisionFinal-GalaxyMorphologyClassifier
-> Branch for testing VGG16
AIG210-Computer Vision Final Project - W2026. A model that classifies images of galaxies into spiral, elliptical, or irregular. The data used is from the GalaxyZoo 2 dataset on Kaggle. The architecture is based on either VGG16 or ResNet18. <br>
Dataset:<br>
https://www.kaggle.com/competitions/galaxy-zoo-the-galaxy-challenge/data -> cite later
